# Quality — V7 Policy Critic

## Overview

This directory contains self-assessment tools for the Policy Critic docs package. The goal is to produce a package that scores ≥ 8/10 by independent AI review before merge.

## Documents

| File | Purpose |
|------|---------|
| `self_scorecard.md` | Internal self-assessment across 10 dimensions |
| `independent_ai_review_packet.md` | Packet for external AI (ChatGPT, etc.) to review and score |
| `acceptance_rubric.md` | Partner acceptance criteria for docs package |

## Scoring Principle

Self-scores are Claude's assessment of the docs. They are NOT independent validation. Independent AI review is **pending** and should be performed by ChatGPT or another separate AI before merge.
