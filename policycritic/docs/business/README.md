# Business — V7 Policy Critic

## Overview

This directory contains the business case, profitability model, and operational strategy for the V7 Policy Critic. These documents answer: "Why should we build this?" and "How much value could it create?"

**Important**: All profitability calculations are **scenario-based estimates**, not promises. Actual results depend on market conditions, execution quality, and empirical validation. No profitability claim can be made before shadow evidence exists.

## Documents

| File | Purpose |
|------|---------|
| `business_plan.md` | Strategic rationale, staged investment, ROI logic, team requirements |
| `profitability_calculation.md` | Scenario-based formulas, break-even analysis, capital allocation |
| `unit_economics.md` | Per-trade economics, cost structure, margin analysis |
| `risk_register.md` | Ranked risks with probability, impact, and mitigations |
| `go_to_market_internal_strategy.md` | Internal rollout, operational readiness, organizational buy-in |

## Key Principle

**No profitability claims without evidence.** The business case describes what COULD happen IF the critic works as designed. Shadow evidence is required before any claim can be made.
